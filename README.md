## Overview

### Description
The app helps users discover cliff jumping spots across the United States through an interactive map that shows different locations and useful information about each one. 
Users can also contribute by adding new cliff jumping spots and rating existing ones, creating a community-driven platform for finding the best places to jump.

### App Evaluation

- **Mobile: This app can use mobile features like location services, maps, and possibly the camera for uploading photos. These features make it strongly mobile-based and not just a simple web app.**
- **Story: The app helps users find cliff jumping locations across different areas, along with important details like height, safety level, and user reviews. It provides a unique and interesting experience for people who enjoy outdoor activities.**
- **Market:The market is more niche compared to other apps, but it still targets adventure seekers and travelers. It’s smaller, but very specific and engaged.**
- **Habit: Users may not use the app every day, but it can still become habit-forming for people who enjoy traveling and exploring new places. They would return to the app when planning trips, checking new locations, or sharing experiences, which creates repeated engagement over time.**
- **Scope: The scope is slightly more challenging because it may involve maps, user-generated content, and more complex data. However, a basic version with simple location listings can still be completed.**

## Product Spec

### 1. User Features (Required and Optional)

**Required Features**

1. Have a interactive map to show all of the cliff jumping locations.
2. Have a login page, where user logs in to access content.


**Optional Features**

1. Have ability to add cliff jumping spots for others to view.
2. Feature to rate the spots with 1-5 stars



### 2. Screen Archetypes

- Login/Register
  - Have a login page, where users logs in to access content.
- Detail
  - The app helps users find cliff jumping locations across different areas, along with important details like height, safety level, and user reviews and additions.
- Creation
  - Have ability to add cliff jumping spots for others to view.

### 3. Navigation

**Tab Navigation** (Tab to Screen)

* Map tab
* Profiles tab
* Jumping spots tab/ add locations tab.

**Flow Navigation** (Screen to Screen)

- Main login screen
  - after loging in it will take you to the main screen that has 3 tabs
- Profiles screen
  - One of the tabs will be a profile screen, which you can click on your own profile for info on it that will be flow navigation style.
- Jumping spots tab
  - Once in the spots tab, you will be able to click on add spot that will use flow navigation to take you to a page to add a jumping spot.
